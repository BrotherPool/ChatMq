Instala o JavaSetup8u231
Instala o otp_win64_22.1 do site https://www.erlang.org/downloads, depois baixa e instala o rabbitmq-server-3.8.1 do site https://www.rabbitmq.com/install-windows.html
Vai na pasta C:\Program Files\RabbitMQ Server\rabbitmq_server-3.8.1\sbin pelo cmd como admin e digita esse comando: rabbitmq-plugins enable rabbitmq_management
Depois roda o cliente.jar
